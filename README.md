# Digital Electronics & Computer Architecture
This repository contains a collection of hardware logic and digital circuit designs developed as part of my Computer Science studies. Each project focuses on fundamental building blocks of modern computing, designed and tested using **Logisim**.

## 🚀 Projects Included

### 1. Binary Arithmetic Calculator (Unit 4)
A 4-bit ALU (Arithmetic Logic Unit) capable of addition, subtraction, multiplication, and division.
* **Key Features:** Real-time binary-to-decimal decoding, MUX-based operation selection.
* [View Project Details](./unit4Calculation/)

### 2. PLD Digital Door Lock System (Unit 5)
A security-focused circuit simulating a Programmable Logic Device (PLD) for home or office security.
* **Key Features:** Pin-based authentication, state management, and visual status indicators.
* [View Project Details](./Unit5_PLD_DoorLock/)

## 🛠️ Tools Used
* **Logisim / Logisim-evolution:** For circuit design and simulation.
* **Git/GitHub:** For version control.

## 🏁 Getting Started
1. Install [Logisim-evolution](https://github.com/logisim-evolution/logisim-evolution).
2. Clone this repo: `git clone https://github.com/pePiangpi/Digital-Electronics-Computer-Architecture.git`
3. Open the `.circ` files to explore the logic.
